# Desafio-MaquinaVirtual-Azure
Desafio DIO sobre máquina virtual Azure.


1. Acessar o portal Azure
2. No menu, selecionar a opção Máquinas virtuais
3. Cricar em Criar
4. Opção Máquina virtual
5. Colocar os detalhes da instância (Nome, Região, Disponibilidade, Segurança, imagem, etc)
6. Avançar para guia Discos e selecionar Tamanho, tipo, volume
7. Avançar para guia Rede, selecionar, Rede virtual, Sub-rede
8. Avançar para guia Gerenciamento, selecionar as opções necessárias
9. Avançar para guia Monitoramento, selecionar as opções necessárias
10. Avançar para guia Avançado, selecionar as opções necessárias
11. Avançar para guia Marcar, selecionar as opções necessárias
12. Avançar para guia Revisar + criar, revisar, e criar.
13. Na implantação da Maquina virtual, foi gerado 4 recursos para o funcionamento correto da máquina (Rede Virtual VNET, Grupo de Segurança NSG, Endereço de IP, e Interface de Rede)
